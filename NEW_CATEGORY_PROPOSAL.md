# SilverMind 새로운 게임 카테고리 제안

| 카테고리 | 게임 수 | 비율 |
|---------|---------|------|
| Memory | 7개 | 26% |
| Spatial | 7개 | 26% |
| Logic | 5개 | 18% |

| Reaction | 4개 | 15% |

---

## 🎯 제안 1: **Attention & Focus** (주의력/집중력)

### 이유
- 현재 Reaction 카테고리가 빠른 반응 속도에 집중
- **지속적 집중력(Selective Attention, Sustained Attention)**은 별도 영역
- 현대인의 스마트폰 사용으로 인한 집중력 저하 문제에 대응

### 게임 아이디어 (5개)

| 게임 이름 | 설명 | 난이도 |
|-----------|------|--------|
| **Spotlight** | 시야가 좁은 스포라이트로 숨겨진 물체 찾기 | Easy~Master |
| **Find Difference** | 두 이미지의 차이점 찾기 (시간 제한) | Easy~Master |
| **Selective Search** | 방해 요소 속에서 특정 타겟 찾기 (Stroop 테스트) | Easy~Master |
| **Concentration Grid** | 1-100 숫자를 순서대로 빠르게 찾기 | Easy~Master |
| **Focus Tracker** | 여러 움직이는 물체 중 하나만 추적 | Easy~Master |

### 인지 능력
- Selective Attention (선택적 주의)
- Sustained Attention (지속적 주의)
- Divided Attention (분할 주의)

---

## 🎯 제안 2: **Language & Verbal** (언어/언어적 능력)

### 이유
- 한국어 + 영어 이중 언어 지원 활용 가능
- **단어 기억(word-recall)**이 있지만, 언어적 처리 능력은 별도
- 어휘력, 언어적 유연성, 언어적 추론 향상

### 게임 아이디어 (5개)

| 게임 이름 | 설명 | 난이도 |
|-----------|------|--------|
| **Word Scramble** | 섞인 글자로 올바른 단어 만들기 | Easy~Master |
| **Anagrams** | 같은 글자로 다른 단어 만들기 | Easy~Master |
| **Synonym Matcher** | 유의어 매칭 (한국어/영어) | Easy~Master |
| **Idiom Riddle** | 속담/관용사 뜻 맞추기 | Easy~Master |
| **Word Chain** | 끝말잇기 (타이밍/단어 제약 있음) | Easy~Master |

### 인지 능력
- Vocabulary (어휘력)
- Language Flexibility (언어적 유연성)
- Semantic Processing (의미 처리)

---

## 🎯 제안 3: **Planning & Decision** (계획/의사결정)

### 이유
- Logic 카테고리가 순차적/논리적 추론에 집중
- **전략적 계획, 자원 관리, 최적화**는 별도 영역
- 실생활 의사결정 능력 향상

### 게임 아이디어 (5개)

| 게임 이름 | 설명 | 난이도 |
|-----------|------|--------|
| **Resource Manager** | 제한된 자원으로 목표 달성 (최적화) | Easy~Master |
| **Task Scheduler** | 여러 작업을 효율적으로 스케줄링 | Easy~Master |
| **Path Optimizer** | 최단 경로/최적 경로 찾기 | Easy~Master |
| **Decision Tree** | 여러 선택지에서 최적 의사결정 | Easy~Master |
| **Risk Assessor** | 확률적 상황에서 최적 선택 | Easy~Master |

### 인지 능력
- Planning (계획력)
- Decision Making (의사결정)
- Problem Solving (문제해결)

---

## 🎯 제안 4: **Creativity & Imagination** (창의력/상상력)

### 이유
- 구조화된 문제(Logic/Pattern) 외에 **개방형 문제 해결**
- 발산적 사고(Divergent Thinking) 훈련
- AI 시대에 필요한 창의적 사고력

### 게임 아이디어 (5개)

| 게임 이름 | 설명 | 난이도 |
|-----------|------|--------|
| **Alternative Uses** | 일상 물건의 새로운 용도 생각하기 | Easy~Master |
| **Metaphor Creator** | 두 개념 연결하여 은유 만들기 | Easy~Master |
| **Story Completion** | 주어진 상황으로 창의적 이야기 완성 | Easy~Master |
| **Shape Transformation** | 도형을 변형하여 새로운 의미 부여 | Easy~Master |
| **Word Association Creative** | 연관 단어 찾기 (창의적/비일상적) | Easy~Master |

### 인지 능력
- Divergent Thinking (발산적 사고)
- Creativity (창의력)
- Imagination (상상력)

## 🎯 제안 5: **Emotional Intelligence** (감성 지능)

### 이유
- 인지 능력 외에 **정서적 인지** 중요
- 표정, 상황, 관점 이해 능력
- 사회적 기술 향상

### 게임 아이디어 (5개)

| 게임 이름 | 설명 | 난이도 |
|-----------|------|--------|
| **Emotion Recognition** | 표정에서 감정 읽기 | Easy~Master |
| **Perspective Taking** | 다른 관점에서 상황 이해하기 | Easy~Master |
| **Social Cues** | 사회적 신호/행동 의미 파악 | Easy~Master |
| **Empathy Scenario** | 타인의 감정 이해/반응 선택 | Easy~Master |
| **Emotion Word Match** | 감정 표현 매칭 (한국어/영어) | Easy~Master |

### 인지 능력
- Emotional Recognition (감정 인식)
- Empathy (공감 능력)
- Social Cognition (사회적 인지)
